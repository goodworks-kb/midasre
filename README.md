# Midas Realty Website

A modern, responsive website for Midas Realty - a real estate company.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Property Listings**: Showcase featured properties with detailed information
- **Service Information**: Comprehensive display of real estate services
- **Contact Form**: Easy-to-use contact form for client inquiries
- **Smooth Navigation**: Fixed navigation bar with smooth scrolling

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)

## File Structure

```
MidasRE/
├── index.html      # Main HTML file
├── styles.css      # All styling
├── script.js       # JavaScript functionality
└── README.md       # This file
```

## Getting Started

1. Open `index.html` in a web browser
2. No build process or dependencies required - it's ready to use!

## Sections

- **Home**: Hero section with call-to-action buttons
- **Properties**: Featured property listings
- **About**: Company information and statistics
- **Services**: Comprehensive service offerings
- **Contact**: Contact information and inquiry form

## Customization

You can easily customize:
- Colors: Modify CSS variables in `styles.css` (`:root` selector)
- Properties: Update the `properties` array in `script.js`
- Content: Edit text directly in `index.html`
- Contact Information: Update contact details in the contact section

## Browser Support

Works on all modern browsers:
- Chrome
- Firefox
- Safari
- Edge
