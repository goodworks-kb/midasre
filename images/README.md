# Images Folder

Place Tina Kim's professional photo here.

**File name:** `tina-kim.jpg` (or `.png`)

**Recommended specifications:**
- Format: JPG or PNG
- Size: At least 600x600 pixels (square format works best)
- File size: Under 500KB for web optimization

**To add the photo:**
1. Download the image from midasre.com or use a professional headshot
2. Save it as `tina-kim.jpg` in this folder
3. If using a different filename, update the `src` attribute in `index.html` (line ~110)

**Alternative:** If the image is hosted online, you can update the `src` attribute in `index.html` to use the full URL instead.
